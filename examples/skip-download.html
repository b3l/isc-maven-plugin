Nightly builds are not necessarily available indefinitely, and you may find from time to time that you need to use a version that has been superseded by a newer release or is otherwise no longer found on the builds page. In this case, the plugin can be made to look locally for a distribution:

If necessary, create a directory that looks like one the plugin would have created for you.

mkdir /tmp/SmartGWT/PowerEdition/4.1d/2013-11-25/zip

Drop a copy of the file/s in question where the plugin can find it.

cp /downloads/smartgwtee-4.1d_11-25-2013.zip /tmp/SmartGWT/PowerEdition/4.1d/2013-11-25/zip

Use the skipDownload property to cause resolution from the local file system.

mvn isc:install -Dproduct=SMARTGWT \
                -Dlicense=EVAL \
                -DbuildNumber=4.1d \
                -DbuildDate=2013-11-25 \
                -Dworkdir=/tmp \
                -DskipDownload=true
